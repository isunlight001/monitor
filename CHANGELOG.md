# 更新日志

## [1.0.0] - 2024-12-03

### 新增功能

#### 基金监控系统
- **基金数据爬取**
  - 实现从http://www.dayfund.cn自动抓取基金净值数据
  - 支持指定日期范围的数据获取
  - 支持增量更新（最近1个月数据）
  
- **基金数据存储**
  - 设计基金净值数据表结构
  - 实现MyBatis Mapper接口
  - 支持批量插入和去重更新

- **监控规则引擎**
  - 规则A: 连续4天或以上上涨/下跌监控
  - 规则B: 单日涨跌幅绝对值≥5%监控
  - 规则C: 连续2-3天累计涨跌幅绝对值≥5%监控
  - 实时计算累计涨跌幅并触发预警

- **邮件通知系统**
  - 集成Spring Mail实现邮件发送
  - 支持预警内容自动生成
  - 包含基金名称、涨跌天数、涨跌幅、净值日期等信息

- **定时任务调度**
  - 实现每晚11点自动执行数据抓取
  - 自动执行监控检查
  - 异常时自动发送邮件预警

- **Web管理界面**
  - 提供基金数据抓取功能
  - 支持手动执行监控检查
  - 提供基金净值数据查询
  - 友好的用户交互界面

#### 线程池监控系统
- **线程池配置管理**
  - 实现自定义线程池配置
  - 支持多种线程池类型配置
  
- **监控指标收集**
  - 集成Spring Boot Actuator
  - 集成Micrometer Prometheus监控
  - 实时收集线程池活跃度、队列大小等指标

- **监控API接口**
  - 提供线程池状态查询接口
  - 提供详细的性能指标接口

#### 通知系统
- **邮件通知服务**
  - 支持简单文本邮件发送
  - 支持HTML格式邮件发送
  - 完善的配置管理和异常处理

- **微信通知服务**
  - 支持Server酱微信推送
  - 支持企业微信通知
  - 统一的通知发送接口

- **统一通知接口**
  - 提供统一的通知服务接口
  - 支持多种通知方式组合发送
  - 完善的服务状态检查机制
