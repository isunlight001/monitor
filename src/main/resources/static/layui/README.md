# Layui版本前端界面说明

## 简介
本目录包含了使用Layui框架重构的前端界面，保持与原有后端API的兼容性，提供更加现代化和统一的用户界面体验。

## 特性
- 使用Layui框架构建，界面美观大方
- 响应式设计，适配不同屏幕尺寸
- 统一的导航菜单和布局结构
- 丰富的表格组件和表单控件
- 集成原有后端API，功能完整

## 页面列表
- `index.html` - 主页/仪表板
- `login.html` - 登录页面
- `fund-monitor.html` - 基金监控页面
- `alert-records.html` - 告警记录页面
- `stock-management.html` - 股票管理页面

## 使用方法
1. 直接访问对应的HTML文件即可使用Layui版本的界面
2. 所有页面保持与原有后端API的兼容性
3. 登录验证逻辑与原版本一致

## 注意事项
- 本版本为前端界面重构，后端API保持不变
- 如需修改功能逻辑，请参考原有页面实现
